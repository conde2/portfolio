Slotrium

Solo Passion Project · Godot · C# · 2024

A roguelike slot-machine RPG built solo in Godot with C# every spin is a turn. Reel results trigger synergistic symbol effects, and between rounds players draft new symbols, relics and modifiers to build runs that snowball, in the spirit of the deck-building roguelike wave (think Luck be a Landlord / Balatro).

The most challenging part was the systems design. I built a flexible, data-driven system for game mechanics, rewards, progression, and events that made it easy to add new content without changing the core code. Although the project was never released, it was my deepest exploration of game design and progression systems.

  • Godot
  • C#
  • Roguelike systems
  • Procedural RNG
  • Data-driven design
